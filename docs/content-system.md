# 内容系统

## 总体结构

当前副业不是单一内容号，而是围绕 **AI 工作流教程** 搭建的内容与转化体系。

| 渠道 | 角色 | 内容形态 |
|------|------|------|
| B站 | 核心流量入口 | 3 分钟扫盲视频、ComfyUI 实操、模型观察、案例拆解 |
| YouTube | 长期搜索沉淀 | 同步 B站长视频 |
| 公众号 | 免费图文沉淀 | 教程图文、工具资讯、模型扫盲、复盘 |
| GitHub | 公开资产沉淀 | prompt、skills、工作流说明、资料索引 |
| RunningHub | 云端工作流承接 | 案例工作流、云端运行、轻植入 |
| 知识星球 | 深度教程和答疑 | 系统课、专题拆解、资料库、答疑陪跑 |
| 夸克网盘 | 资料包承接 | 整合包、模型资料、工作流包 |

## 内容主线

### 1. AI 工作流扫盲

目标：给新手建立大地图。

代表选题：

- 现在 AI 生图到底分几种玩法？
- 国产/开源生图模型这么多，新手到底看哪几个？
- 为什么现在大家都在说工作流？
- 本地部署和云端平台怎么选？
- 新手想做 AI 图片/视频，第一步该学什么？

### 2. ComfyUI 实操入门

目标：从“能跑通”走到“能理解”。

代表选题：

- ComfyUI 入门：不要一上来就啃节点，先理解这 5 件事
- 工作流导入后跑不起来，先检查这几个地方
- KSampler、模型路径、VAE、ControlNet 到底怎么理解
- AI 短剧角色图工作流，新手如何从 0 跑通

### 3. AI 视频与分镜

目标：把 AI 视频从“抽卡生成”变成“分镜生产”。

代表选题：

- AI 视频创作，正在进入可控协作时代
- Seedance 2.0 分镜 prompt 怎么写
- 首尾帧、角色一致性、镜头运动怎么拆
- 为什么 AI 视频会闪烁、变脸、崩画

### 4. 提示词资产

目标：把一次性 prompt 变成可复用资产。

代表选题：

- 一个好 prompt 不只是提示词，而是一张资产卡片
- 如何管理 AI 生图提示词库
- 如何把 GPT 人设整理成可复用 skill
- 提示词版本、标签、场景和变量怎么设计

## 单条视频结构

每条视频约 3 分钟：

1. 0-10 秒：痛点开场
2. 10-40 秒：解释一个关键概念
3. 40-140 秒：案例或界面演示
4. 140-165 秒：常见误区
5. 165-180 秒：总结、下期预告、关注理由

## 公众号文章结构

公众号不强行做单一连载，而是两条线穿插：

- 扫盲系列：建立长期教程资产
- 资讯/工具观察：承接热点、模型和平台变化

文章默认结构：

1. 用新手问题开场
2. 先给核心判断
3. 分 3-6 个小节讲清楚
4. 少堆术语，多用场景解释
5. 结尾引导关注系列和资料沉淀
