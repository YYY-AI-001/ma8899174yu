# 源力魔方 IP 定位

## 一句话定位

**源力魔方 = 面向新手和内容创作者的 AI 工作流研究所。**

核心不是追热点模型，而是把 AI 生图、ComfyUI、AI 视频、提示词和云端平台，整理成可理解、可复用、可实践的工作流。

## 核心人设

| 维度 | 设定 |
|------|------|
| 角色 | AI 工作流扫盲者 / ComfyUI 实操拆解者 / AI 视频分镜研究者 |
| 气质 | 清楚、耐心、实用，不故弄玄虚 |
| 目标用户 | AI 生图新手、ComfyUI 初学者、AI 视频创作者、内容副业探索者 |
| 主要价值 | 帮用户从“工具很多，不知道怎么学”走到“能跑通、能理解、能复用” |
| 差异化 | 不只发效果图，而是解释效果背后的模型、节点、提示词和工作流 |

## 内容边界

做：

- AI 工作流扫盲
- ComfyUI 实操入门
- AI 生图模型观察
- AI 视频分镜和提示词
- RunningHub / 云端工作流演示
- 提示词资产管理
- 可复用资料和 skills

暂不做：

- 纯新闻搬运
- 泛 AI 鸡汤
- 和主线无关的实验项目
- 只展示效果、不解释流程的内容

## 表达风格

- 先讲人话，再讲术语。
- 先给判断，再给例子。
- 少用“最强”“封神”“颠覆”这类词。
- 把复杂概念拆成新手能理解的分类。
- 每篇内容都尽量回答一个具体问题。

## 标准结尾方向

```text
后面我会继续整理 AI 工作流、ComfyUI、AI 视频和提示词资产相关内容，包括模型怎么选、工作流怎么跑、导入后报错怎么排查，以及适合新手练习的案例。

如果你也想系统学习，可以先关注。
```
