# 承接链路

## 北极星目标

短期目标：**B站达到 3000 粉**。

当前策略：

- 每周 2 条视频
- 内容系列化
- 统一标题、封面、简介、置顶评论和资料承接
- 公众号和知识星球围绕视频做延伸
- RunningHub 前期轻植入，不硬推

## 从内容到转化

```text
B站/YouTube 视频
  -> 公众号图文教程
  -> GitHub 公开资料和 skills
  -> 夸克网盘资料包
  -> RunningHub 工作流学习
  -> 知识星球系统教程和答疑
```

## B站置顶评论模板

```text
本期图文教程：公众号「源力魔方」
新手资料包/整合包：见置顶资料链接
工作流学习：RunningHub 主页
系列教程/答疑：知识星球

我会持续更新 AI 工作流扫盲、ComfyUI 实操和 AI 视频分镜系列。想系统学可以先关注。
```

## GitHub 的角色

GitHub 不做泛泛代码展示，而是作为公开资产库：

- 存放 Codex Skills
- 存放 prompt 资产模板
- 存放公开教程索引
- 存放工作流说明文档
- 作为公众号和视频的长期资料引用入口

## 资料包 v0.1 建议

轻量整合包优先包含：

- 新手学习路线图
- ComfyUI 常见目录说明
- 常见报错检查清单
- AI 生图模型分类表
- Prompt 资产卡片模板
- Seedance 分镜 prompt 模板

## 月度复盘指标

每周记录：

- 视频数量
- 公众号数量
- 星球更新数量
- B站播放、收藏、评论、粉丝变化
- YouTube 播放和订阅变化
- 网盘转存/下载收益
- RunningHub 推广数据
- 精力状态 1-10 分

每月判断：

- 是否保持每周 2 条视频
- 粉丝是否按 3000 粉目标推进
- 是否有稳定评论、私信或咨询
- 整合包是否带来转存和下载
- 知识星球是否有新增和互动
- 是否需要调整内容方向
