# Codex Skills

这个目录收纳源力魔方正在使用和迭代的 Codex Skills。

## Skill 列表

| Skill | 说明 |
|------|------|
| [prompt-asset-director](./prompt-asset-director/SKILL.md) | 将零散提示词整理、优化、分类并沉淀为可复用资产 |
| [seedance-storyboard-director](./seedance-storyboard-director/SKILL.md) | 为 Seedance 2.0 / AI 视频创作设计分镜、角色一致性和提示词 |

## 使用方式

将对应 skill 目录复制到本地 Codex skills 目录即可，例如：

```text
~/.codex/skills/prompt-asset-director/
~/.codex/skills/seedance-storyboard-director/
```

然后在 Codex 中用 `$skill-name` 调用。
